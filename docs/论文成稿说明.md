# 论文成稿文件说明

## 生成的 Word 文件

| 文件 | 说明 |
|------|------|
| `MathorCup_C_paper_final.docx` | **完整论文主稿**（摘要—正文—图表—附录），由 `scripts/build_mathorcup_paper_docx.py` 自动生成，数值与 `outputs/run_20260417_211020/` 对齐。 |
| `MathorCup_2026_paper_working.docx` | 将官方 `2026年MathorCup数学应用挑战赛论文模板.doc` **转换** 得到的 `docx` 工作副本（仅含题目、摘要占位、目录占位等原始版式）。 |

## 使用建议

1. **直接提交**：若竞赛允许自定版式，可直接使用 `MathorCup_C_paper_final.docx`。  
2. **套模板版式**：若必须完全沿用官方模板页眉页脚，可将 `MathorCup_C_paper_final.docx` 中正文全文复制到 `MathorCup_2026_paper_working.docx` 正文起始位置（替换“（摘要正文）”“（这里开始论文正文）”等占位），再统一调整字体与行距。  

## 重新生成

```bash
pip install python-docx
python scripts/build_mathorcup_paper_docx.py
```

## 章节与结果映射

见 `docs/论文章节与结果映射.md` 与根目录 `outputs_Index.md`。
